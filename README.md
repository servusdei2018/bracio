# bracio [WIP]

A high-performance, drop-in replacement for the Python asyncio event loop, powered by Rust's Tokio runtime.

> [!WARNING]  
> `bracio` is pre-alpha software and is currently under active development. It is not yet ready for production use. The API is unstable and subject to breaking changes without prior notice. Use at your own risk.
